All-in-One Dashboard

This project is a Next.js application providing live market data for:
- PSX (Pakistan Stock Exchange) stocks via scraping
- NASDAQ stocks via Yahoo Finance and NASDAQ screener

Features:
- Card and table views with pagination, sorting, filters (sector/index/category)
- Watchlists management (create, add, remove)
- Symbol detail pages with interactive charts
- Global settings context for refresh interval and theme
- Sidebar navigation including Dashboard, PSX, NASDAQ, Watchlist, Metals,
  Forex, Crypto, Expenses, Settings

APIs:
- /api/psx-stocks - scrapes data from PSX website
- /api/nasdaq-stocks - fetches symbol list from Nasdaq, quotes from Yahoo
- /api/watchlists - CRUD for user watchlists
 - /api/tts - ElevenLabs Text-to-Speech proxy (requires `ELEVENLABS_API_KEY`)

Styling: Tailwind CSS with dark mode

Components: StockCard (exchange-aware), Sidebar, ThemeProvider

See PROJECT_DOCUMENTATION.md for detailed breakdown.
